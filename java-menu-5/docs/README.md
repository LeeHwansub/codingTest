# 기능 목록

## 1. 입력 기능
- [x] 코치 이름 입력 받기 (쉼표로 구분)
- [x] 코치 이름 유효성 검증 (2-4글자, 2-5명)
- [x] 각 코치의 못 먹는 메뉴 입력 받기
- [x] 못 먹는 메뉴 유효성 검증 (최대 2개)

## 2. 출력 기능
- [x] 시작 문구 출력
- [x] 입력 안내 문구 출력
- [x] 메뉴 추천 결과 출력 (표 형식)
- [x] 완료 문구 출력
- [x] 에러 메시지 출력

## 3. 메뉴 추천 로직
- [x] 카테고리 목록 관리 (일식, 한식, 중식, 아시안, 양식)
- [x] 각 카테고리의 메뉴 목록 관리
- [x] 주간 카테고리 추천 (같은 카테고리 최대 2회)
- [x] 일일 메뉴 추천 (코치별 중복 방지, 못 먹는 메뉴 제외)
- [x] 랜덤 카테고리 선택
- [x] 랜덤 메뉴 선택

## 4. 도메인 모델
- [x] Coach 클래스 (이름, 못 먹는 메뉴 목록)
- [x] Category enum (일식, 한식, 중식, 아시안, 양식)
- [x] MenuRecommendation 클래스 (주간 추천 결과 관리)

## 5. 예외 처리
- [x] IllegalArgumentException 처리
- [x] 에러 메시지 포맷팅
